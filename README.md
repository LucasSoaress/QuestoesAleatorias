# Questoes Aleatorias
Respostas para duas questões de uma prova

```
Faça um algoritmo que leia três notas de um aluno, calcule e escreva a média final deste aluno.
Considerar que a média é ponderada e que o peso das notas é 2, 3 e 5. 
Fórmula para o cálculo da média final é:
                  n1 * 2 + n2 * 3 + n3 * 5
 mediafinal = -----------------------------------
                              10 
```

```
Desenvolva uma solução algoritmica que dada a média de um aluno, identifique se o mesmo foi aprovado
(caso a média seja maior ou igual a 6.0) ou reprovado (caso a nota seja inferior a 6.0)
```
